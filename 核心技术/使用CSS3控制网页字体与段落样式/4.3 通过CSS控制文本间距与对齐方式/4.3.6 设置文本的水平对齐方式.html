语法格式：{text-align: sTextAlign}

start: 文本向行的开始边缘对齐

end: 文本向行的结束边缘对齐

left: 文本向行的左边缘对齐。在垂直方向的文本中，文本在 left-to-right 模式下开始边缘对齐

rigth: 文本向行的右边缘对齐。在垂直方向的文本中，文本在 right-to-left 模式下结束边缘对齐

center: 文本早行内居中对齐

justify: 文本根据 text-justify 的属性是指方法分散对齐。即两端对齐，均匀分布

match-parent: 继承父元素的对齐方式，但有个例：继承的 start 或 end 值是根据父元素的 direction 值进行计算的，因此计算的结果可能是 left 或 right

“string”: string 是一个单个的字符，否则，就忽略此设置。按指定的字符进行对齐。此属性可以跟其他关键词同时使用，如果没有设置字符，则默认值是 end 方式

inherit: 继承父元素的对齐方式
